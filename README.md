# StratRoulette.tf

Source code for [StratRoulette.tf](https://stratroulette.tf), a Team Fortress 2
version of [StratRoulette.com](www.stratroulette.com). Made with [Elm](http://elm-lang.org/).

## Install
```
make install
```
Requires `npm` and `bower`.

## Build
```
make
```

## Development
```
watch make dev
```

## Deploy
```
make deploy
```
Remember to chance the application ID in `app.yaml`.
